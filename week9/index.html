The Window Object

The Browser Object Model

A collection of properties and methods that contain information about the browser and computer screen

Dialogs
  • Alert - Calls browser 'alert' object
  • Confirm - Pauses browser processes and asks for permission to continue
  • Prompt - A browser object used to collect input

Browser Information

  • Navigator - An object that contains information about the browser
      Properties:
    • navigator
    • userAgent
  • Location - Contains informations about the URL
      Properties:
    • href - Returns URL as a string
    • protocol - Returns the connection protocol
    • host - Returns string describing the domain
    • hostname - Returns string describing the domain
    • port - Returns string describing the port number
    • pathname - Returns pathname following domain as a string
    • search - Returns a stringified query
    • hash - Places a pound sign after an identifier
    • origin - Shows the origin diomain and path
    • reload - Forces a reload
    • assign - Load another resource from a URL
    • replace - Same as the above, but saved in session history
    • toString - Returns string with whole URL

Cookies

Cookies are files that are stored locally on the computer. They are used to let the browser remember user data between requests.

Use the document.cookie property

Timing Functions

  • setTimeout
  • setInterval

Animation

The two previously mentioned timing functions can be used to create animations inn conjunction with the html and css.

Use requestAnimationFrame

HTML5 APIs

The 'data-' Attribute

This attribute is used to add custom data to a web page. These are private, so they are only meant to be used by javascript and not third-party services.

HTML5 APIs

HTML APIs allow browsers to access data from computer hardware.

HTML Web Storage

Similar to cookies, the Web Storage API is easier to use and can store more data. It allows data to be remembered inbetween sessions without forcing data-sharing with a server. It is also uniquee because session data can be created and editied from multiple windows simutaneously.

An Extra Note on APIs

There are thousands of APIs that help in a wide variety of different circumstances. Many are listed in the text, including APIs for video controls, notifications, websockets, and more.

Drawing With Canvas

Canvas is an element used to create an environment for javascript drawing instruction/printing.

Just about every element on the canvas is customizable... Height, width, background color, and on top of that every drawn element can also be edited and created with a high level of control.

3d rendering is only possible with third-party providers.

Shims and Polyfills

Shims and polyfills are what allow your implemented versions of APIs to continue to function properly in the extremely fast-changing world of the internet.

Backwards compatability as well as forward is a big deal in web development, and with shims and polyfills, you can ensure that your application works on all browser environments.
